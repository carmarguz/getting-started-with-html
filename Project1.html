Main takeaways

Web Page

A web page is a text document written in a language called HTML ( Hyper Text Transfer Protocol )

How Coding Works

Coding happens when a programmer, in a language a computer can understand, tells the PC what to do.

Computers are stupid....or are they?

Sintax is very important when communicating with a computer. One will receive the wrong information displayed if the request/order is wrong.
This can be frustrating at times. However, on the other hand, if the programmer knows how to "talk" correctly to the computer...the computer will do anything the programmer wants!

It's not easy to remember everything

One does not necessarily need to remember everything. There are plenty of online resources to research any particular tag.

Basic HTML vocabulary

It is important to feel comfortable with the jargon

  • Tag

    • An HTML tag is always contained within angled brackets {}

    Most tags have an opening tag ( "" for example) and a closing tag ( ).

    There are some tags, called "void" tags, which do not require closing ( like the
    tag.)

  • Element

    • Everything within a set of opening and closing tags is considered an Element.

  • Attribute

    • the property of an HTML element is called an attribute.
